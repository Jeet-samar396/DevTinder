# DevTinder 🚀

DevTinder is a full-stack web application that allows developers to connect, match, and chat with each other based on shared interests and skills. It provides a Tinder-like experience tailored for developers.

## 🔥 Features

* User Authentication (Signup/Login)
* Swipe-based Developer Matching
* Real-time Chat System
* Profile Creation & Editing
* Secure Backend with JWT Authentication

## 🛠️ Tech Stack

**Frontend:**

* React (Vite)
* Tailwind CSS

**Backend:**

* Node.js
* Express.js
* MongoDB

**Other:**

* JWT Authentication
* REST APIs

## ⚙️ Installation

### 1. Clone the repository

```
git clone <your-repo-link>
cd devtinder
```

### 2. Setup Backend

```
cd server
npm install
```

Create a `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
```

Run backend:

```
npm start
```

### 3. Setup Frontend

```
cd client
npm install
npm run dev
```
